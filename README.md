# ModernFrontendEntry_HIU
モダンフロントエンド入門の資料リポジトリ

## 前提知識・準備物

この講座は以下の知識があることを前提に進めます。  
事前学習で身に付けられる見込みがあれば参加して大丈夫ですが、基本的には下記スキルを身に付けられる講義を履修している前提です。

- タグやプロパティやDOM操作などのHTML/CSS,JavaScriptの知識
- ファイルやディレクトリなどのコンピュータの基礎知識
- URLやクライアントサーバモデルなどのインターネットの基礎知識
- 変数、定数、配列、オブジェクト、関数、引数、返り値などのプログラミングの基礎知識
- Gitのadd、commit操作、GitHubへのpushを事前に学習しておくこと

事前学習資料の一例を挙げておきます。  
他の資料を用いて事前学習を行ってももちろん大丈夫です。

- Node.js入門: [とほほのNode.js入門 5.Hello World(コンソール版)まで](https://www.tohoho-web.com/ex/nodejs.html)
- Git、GitHub入門: [サル先生のGit入門](https://backlog.com/ja/git-tutorial/)

また、この講座は以下のものを必須とします。

- PC(タブレット不可、Chromebook非推奨)
- GitHubアカウント：個人のもの、Repositoryは講座内で作成する
- Figmaアカウント：個人のもの

自身のコンピュータにはGitをインストールし、自身のGitHubアカウントと連携しておいてください。  

Windowsのインストールは[こちら](https://qiita.com/takeru-hirai/items/4fbe6593d42f9a844b1c)を参考にしてください。  

macOSのインストールは以下のとおりです。

```bash
brew install git
```

Linuxのインストールの例は以下のとおりです。

```bash
sudo apt-get install git
```
