モダンフロントエンド入門の資料リポジトリ

## 前提知識・準備物

この講座は以下の知識があることを前提に進めます。  
事前学習で身に付けられる見込みがあれば参加して大丈夫ですが、基本的には下記スキルを身に付けられる講義を履修している前提です。

- タグやプロパティやDOM操作などのHTML/CSS,JavaScriptの知識
- ファイルやディレクトリなどのコンピュータの基礎知識
- URLやクライアントサーバモデルなどのインターネットの基礎知識
- 変数、定数、配列、オブジェクト、関数、引数、返り値などのプログラミングの基礎知識
- Gitのadd、commit操作、GitHubへのpushを事前に学習しておくこと

事前学習資料の一例を挙げておきます。  
他の資料を用いて事前学習を行ってももちろん大丈夫です。

- Node.js入門: [とほほのNode.js入門 5.Hello World(コンソール版)まで](https://www.tohoho-web.com/ex/nodejs.html)
- Git、GitHub入門: [サル先生のGit入門](https://backlog.com/ja/git-tutorial/)

また、この講座は以下のものを必須とします。

- PC(タブレット不可、Chromebook非推奨)
- GitHubアカウント：個人のもの、Repositoryは講座内で作成する
- Figmaアカウント：個人のもの

自身のコンピュータにはGitをインストールし、自身のGitHubアカウントと連携しておいてください。  
アカウント連携は[`gh`コマンド](https://qiita.com/s_yasunaga/items/110d21828bd4f578850d)が楽です。

Windowsのインストールは[こちら](https://qiita.com/takeru-hirai/items/4fbe6593d42f9a844b1c)を参考にしてください。  

macOSのインストールは以下のとおりです。

```bash
brew install git gh
```

Linuxのインストールの例は以下のとおりです。

```bash
sudo apt-get install git gh
```

# 講座計画

本講座の計画は以下の通りです。

### 1. React.js(Next.js)の特徴、利点、欠点、TypeScritpt との組み合わせ

現在主流のライブラリ「React.js」とそのフレームワーク「Next.js」について解説します。  
React の特徴、利点、欠点について理解しましょう。  
また、JavaScript のスーパーセットである「TypeScript」について学びましょう。

#### 目標

-   React.js がどういう存在か理解する
-   React.js と Next.js の違いについて理解する
-   TypeScript の存在と利点について理解する

#### 成果物

TypeScript でコードを書いてみよう。


### 2. NPM の使い方、JSX 記法、コンポーネント設計

React を使う上で必須となる「NPM」について解説します。  
また、React に採用されている「JSX 記法」と、React の特徴である「コンポーネント設計」についても解説します。

#### 目標

-   NPM の使い方を覚える
-   JSX 記法について理解する
-   コンポーネントの概念を理解する
-   React の特徴について理解する


### 3. React Hook と CSS

React コンポーネントのロジックにおいて重要なHookについて解説します。
また、React における CSS の扱い方について解説します。

#### 目標

-   React Hook について理解する
-   CSS フレームワークの導入方法について理解する
-   メジャーな CSS フレームワークについて知る


### 4. ページ遷移

ページ遷移の実装方法について、React と Next での違いに触れながら解説します。加えて、ページ遷移の実現方法についても解説します。

#### 目標

-   ページ遷移の実装の特徴について理解する
-   ページ遷移の実現方法である SPA、MPA と CSR、SSR について理解する

#### 成果物

ReactとNextでページ遷移のサンプルをつくろう。


### 5. WebAPI とサーバ関数

API についてと JavaScript で API を実行する方法について解説します。  
また、API に代わる手段のひとつとして、サーバ関数について解説します。

#### 目標

-   API の概念について理解する
-   WebAPI と API の関係性について理解する
-   JavaScript での WebAPI の使い方について理解する
-   サーバ関数の利点と使い方を理解する


### 6. サーバコンポーネントとクライアントコンポーネント

Next.js と App Router の組み合わせによって利用できるサーバコンポーネントとクライアントコンポーネントについて解説します。

#### 目標

-   サーバコンポーネントとクライアントコンポーネントの特徴と利点について理解する
-   サーバコンポーネントとクライアントコンポーネントそれぞれにしかできないことを理解する
-   SPA や MPA との関係について理解する

#### 成果物

サーバアクション、サーバコンポーネント、クライアントコンポーネントのサンプルをつくろう。
